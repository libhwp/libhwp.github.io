HWP(5.0) Format Library

“본 라이브러리는 한글과컴퓨터의 HWP 문서 파일(.hwp) 공개 문서를 참고하여 개발하였습니다.”

시작하기

접근성

Java, 혹은 Kotlin을 사용하시나요? 아니면 JVM(Java Virture Machine)을 사용하는 언어(Clojure, Groovy, Scala)나 심지어 Jython, JRuby를 사용하신다면, 바로 사용하세요!

유연성

일반 프로젝트, Spring Framework, GUI, Mobile (Android / IOS)등 다양한 분야에 적용할 수 있습니다.

멀티 플랫폼

Windows, Linux, Mac OSX, Android, Raspberry pi등 JVM을 지원하는 어떠한 운영체제든 같은 결과물을 도출할 수 있습니다.

# 시작하기

Released JAR

  • GitHub Release Link

gradle

dependencies {
    compile group: 'com.tang.tools', name: 'libhwp', version: '1.x.x' // 최신 버전
}

maven

<dependency>
    <groupId>com.tang.tools</groupId>
    <artifactId>libhwp</artifactId>
    <version>1.x.x</version> <!-- 최신버전 -->
</dependency>

Source Code

git clone https://github.com/accforaus/libhwp.git # fork repository

cd libhwp

./gradlew clean build

WARNING

libhwp는 한글 포맷(5.0)에 맞춰져 있습니다. 그러므로 이 라이브러리는 2000년 10월 이후에 출시된 한글 제품군(한글 워디안, 한글 2002, 한글 2005, 한글 2007, 한글 2010, 한글 2014, 한글 2018등)에서 사용가능하며, 문서 버전에 따라 큰 골격은 유지되나, 추가적인 정보들에 의해 약간의 차이가 있습니다.
