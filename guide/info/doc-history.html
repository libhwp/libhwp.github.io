# 문서 이력 관리

한글 메뉴의 파일-문서 이력 관리에서 표시 밒 생성되는 문서의 이력 정보를 저장하는 장소입니다.

이력 정보 데이터를 "DocHistory"라는 새로운 스토리지로 저장합니다. DocHistory 스토리지에 이력 정보 아이템은 추후 추가 정보가 저장되었을 때 하위 호환성을 위하여 다음의 구조로 저장합니다.

문서 이력 정보의 각각의 아이템은 히스토리 혹은 히스토리 아이템이라 하며 한글 Counpund구조 내에서 각 아이템은 DocHistory라는 스토리지 내부에 VersionLog%d(%d는 버전) 이름의 스트림으로 저장됩니다. 또한 각각 아이템은 압축, 암호화되어 저장됩니다.
