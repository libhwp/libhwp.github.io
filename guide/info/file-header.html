자료형 길이 Kotlin / Java 설명

DWORD 4 HWPFileVersion 파일 버전

# 파일 버전

한글 문서의 파일 버전을 나타냅니다. 0xMMNNPPrr의 형태(예 5.0.3.0)로 나타내며, Kotlin의 HWPFileVersion객체 에서 Long 자료형으로 표현됩니다.

  • MM(Short): 문서 형식의 구조가 완전히 바뀌는 것을 나타냄. 숫자가 다르면 구 버전과 호환 불가능
  • nn(Short): 큰 구조는 동일하나, 큰 변화가 있는 것을 나타냄. 숫자가 다르면 구 버전과 호환 불가능.
  • PP(Short): 구조는 동일, Record가 추가되었거나, 하위 버전에서 호환되지 않는 정보가 추가된 것을 나타냄. 숫자가 달라도 구 버전과 호환 가능.
  • rr(Short): Record에 정보들이 추가된 것을 나타냄. 숫자가 달라도 구 버전과 호환 가능.

# 속성 1

각 속성들은 전부 HWPFileHeader객체 안에 boolean값으로 저장되어 지며 아래의 표와 같이 표현됩니다.

범위 자료형 변수 설명
bit 0 Boolean compressed 압축 여부
bit 1 Boolean hasPssword 암호 설정 여부
bit 2 Boolean isDeploymentDocument 배포용 문서 여부
bit 3 Boolean saveScript 스크립트 저장 여부
bit 4 Boolean isDRMDocument DRM 보안 문서 여부
bit 5 Boolean hasXMLTemplate XMLTemplate 스토리지 존재 여부
bit 6 Boolean hasDocumentHistory 문서 이력 관리 존재 여부
bit 7 Boolean hasSignature 전자 서명 정보 존재 여부
bit 8 Boolean encryptPublicCerification 공인 인증서 암호화 여부
bit 9 Boolean savePrepareSignture 전자 서명 예비 저장 여부
bit 10 Boolean isPublicCerticationDRMDocument 공인 인증서 DRM보안 문서 여부
bit 11 Boolean isCCLDocument CCL문서 여부
bit 12 Boolean isMobileOptimize 모바일 최적화 여부
bit 13 Boolean isSecurityPersonalPrivacy 개인 정보 보안 문서
bit 14 Boolean hasTrackChange 변경 추적 문서 여부
bit 15 Boolean hasKOGLcopyright 공공누리(KOGL) 저작권 문서
bit 16 Boolean hasVideoControl 비디오 컨트롤 포함 여부
bit 17 Boolean hasOrderFieldControl 차례 필드 컨트롤 포함 여부
bit 18-31 - - 예약

# 속성 2 new

한글 문서 파일 구조 문서의 업데이트에 따라 파일 인식 정보 안에 새로운 속성값들이 생겼으며, 기존의 값들에서 더 추가된 정보를 가지고 있습니다. HWPKOGLCCLInfo 객체 안에서 표현되며 각 속성 모두 boolean값을 가집니다.

범위 자료형 변수 설명
bit 0 Boolean koglCclInfo CCL, 공공누리 라이선스 정보
bit 1 Boolean isPreventCopy 복제 제한 여부
bit 2 Boolean isPermitCopy 동일 조건 하에 복제 허가 여부
(복제 제한인 경우 무시)
bit 3-31 - - 예약

# EncryptVersion new

마찬가지로 한글 문서 파일 구조 문서의 업데이트에 따라 파일 인식 정보 안에 새로운 속성값들이며, enum class 로 표현되며, HWPEncryptVersionType 객체로 정의 합니다.

값 상수 설명
0 None None
1 LessHWP25 한글 2.5 버전 이하
2 EnhancedHWP30 한글 3.0 버전 Enhanced
3 OldHWP30 한글 3.0 버전 Old
4 MoreHWP70 한글 7.0 버전 이후
