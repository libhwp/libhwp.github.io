# Introduction

Notice

"본 라이브러리는 한글과 컴퓨터의 한글 문서 파일(.hwp) 공개 문서를 참고하여 개발하였습니다."

# 한글 5.0 파일 포맷 라이브러리

libhwp는 프로젝트를 진행하며 맞닥드린 한글 파일의 한계 때문에 제작된 라이브러리 입니다. 자바로 진행된 백앤드 프로젝트에서 hwplib르 사용하였지만, 언어를 Kotlin으로 변경함으로 위의 라이브러리를 Kotlin으로 포팅한 결과물이 현재의 libhwp 입니다. 코어 로직은 큰 변화가 없지만, 많은 부분이 리펙토링되었고, 간결한 Kotlin 문법의 도움을 받아 코드량의 감소도 있었습니다. 계속 필요한 부분은 개발하고 있지만, HWP 파일 포맷 문서의 자료가 완벽하지 않기에 최대한 구현을 했음을 인지하여 주시기 바랍니다.

# [MS-CFB]: Compound File Binary File Format

한글 문서 파일 형식 5.0의 구조는 Microsoft Compound File Binary File Format에 기초하고 있습니다. 그렇기 때문에 OLE2 파일을 읽기 위해 Apache POI Library를 사용하여 한글 파일의 바이너리를 읽고, 쓸수 있습니다.

# 라이브러리에서 제공하는 기능

  • 한글(hwp) 문서 읽기
    • 저장소에 있는 파일 읽기
    • URL상에 있는 파일 읽기
val hwp: HWPDocument = HWPDocument("local_file_paths") // read local file

val hwp: HWPDocument = HWPDocument(urlInstance) // read from URL

// do something

HWPDocument hwpLocal = new HWPDocument("local_file_paths"); // read local file

HWPDocument hwpUrl = new HWPDocument(new URL("url_path")); // read from URL

// do something

  • 한글(hwp) 문서 쓰기
val hwp: HWPDocument = HWPDocument('...') // read document

// do something

hwp.write('write_path') // write document

HWPDocument hwp = new HWPDocument('...'); // read document

// do something

hwp.write('write_path'); // write document

  • 한글(hwp) 문서 복사하기
val hwp: HWPDocument = HWPDocument('...') // read document

val copied: HWPDocument = hwp.copy() // copy document

// do something

HWPDocument hwp = new HWPDocument('...'); // read document

HWPDocument copied = hwp.copy() // copy document

// do something

  • 한글(hwp) 문서 이어 붙히기

val original_hwp: HWPDocument = HWPDocument('...') // will be appended

val target_hwp: HWPDocument = HWPDocument('...') // target document

target_hwp + original_hwp
//or
target_hwp = target_hwp + original_hwp
//or
target_hwp = target_hwp.plus(original_hwp)
//or
val new_hwp: HWPDocument = traget_hwp + original_hwp

target_hwp.addParagraph(some_paragraph, original_hwp) // add paragraph in target_hwp

target_hwp.addParagraphs(some_paragraphs, original_hwp) // add paragraphs in target_hwp

// do somethig

HWPDocument original_hwp = new HWPDocument('...'); // will be appended

HWPDocument target_hwp = new HWPDocument('...') // target document

target_hwp.plus(original_hwp);
//or
target_hwp = target_hwp.plus(original_hwp);
//or
HWPDocument new_hwp = target_hwp.plus(original_hwp);

target_hwp.addParagraph(some_paragraph, original_hwp); // add paragraph in target_hwp

target_hwp.addParagraphs(some_paragraphs, original_hwp); // add paragrahps in target_hwp

WARNING

In Java, target_hwp + original_hwp is not working! Java does not have operator overloading
