# 데이터 레코드

문서 정보(DocInfo)에서 사용되는 데이터 레코드는 다음과 같습니다.

Tag ID Value Kotlin 설명
HWPTAG_DOCUMENT_PROPERTIES HWPTAG_BEGIN DOCUMENT_PROPERTIES 문서 속성
HWPTAG_ID_MAPPINGS HWPTAG_BEGIN + 1 ID_MAPPINGS 아이디 매핑 헤더
HWPTAG_BIN_DATA HWPTAG_BEGIN + 2 BIN_DATA BinData
HWPTAG_FACE_NAME HWPTAG_BEGIN + 3 FACE_NAME Typeface Name
HWPTAG_BORDER_FILL HWPTAG_BEGIN + 4 BORDER_FILL 테두리/배경
HWPTAG_CHAR_SHAPE HWPTAG_BEGIN + 5 CHAR_SHAPE 글자 모양
HWPTAG_TAB_DEF HWPTAG_BEGIN + 6 TAB_DEF 탭 정의
HWPTAG_NUMBERING HWPTAG_BEGIN + 7 NUMBERING 번호 정의
HWPTAG_BULLET HWPTAG_BEGIN + 8 BULLET 불릿 정의
HWPTAG_PARA_SHAPE HWPTAG_BEGIN + 9 PARA_SHAPE 문단 모양
HWPTAG_STYLE HWPTAG_BEGIN + 10 STYLE 스타일
HWPTAG_DOC_DATA HWPTAG_BEGIN + 11 DOC_DATA 문서의 임의의 데이터
HWPTAG_DISTRIBUTE_DOC_DATA HWPTAG_BEGIN + 12 DUSTRUBYTE_DOC_DATA 배포용 문서 데이터
HWPTAG_COMPATIBLE_DOCUMENT HWPTAG_BEGIN + 14 COMPATIBLE_DOUMENT 호환 문서
HWPTAG_LAYOUT_COMPATIBILITY HWPTAG_BEGIN + 15 LAYOUT_COMPATIBILITY 레이아웃 호환성
HWPTAG_TRACKCHANGE HWPTAG_BEGIN + 16 TRACKCHANGE 변경 추적 정보
HWPTAG_MEMO_SHAPE HWPTAG_BEGIN + 76 MEMO_SHAPE 메모 모양
HWPTAG_FORBIDDEN_CHAR HWPTAG_BEGIN + 78 FORBIDDEN_CHAR 금칙처리 문자
HWPTAG_TRACK_CHANGE HWPTAG_BEGIN + 80 TRACK_CHANGE 변경 추적 내용 및 모양
HWPTAG_TRACK_CHANGE_AUTHOR HWPTAG_BEGIN + 81 TRACK_CHANGE_AUTHOR 변경 추적 작성자
