# 바이너리 데이터

Tag ID

HWPTAG_BIN_DATA

# 데이터 표

바이너리 데이터를 나타냅니다.

libhwp에서 HWPBinData객체로 표현됩니다.

자료형 길이 타입 변수 설명
그림, OLE 등의 바이너리 데이터 아이템에 대한 정보
UINT16 2 HWPBinDataProperty property 속성
WORD 2 - - Type이 LINK일 때, 연결 파일의 절대 경로 길이 (len1)
WCHAR array[len1] 2 x len1 String? absolutePathForLink Type이 LINK일 때, 연결 파일의 절대 경로
WORD 2 - - Type이 LINK일 때, 연결 파일의 상대 경로 길이 (len2)
WCHAR array[len2] 2 x len2 String? relativePathForLink Type이 LINK일 때, 연결 파일의 상대 경로
UINT16 2 Int binDataID Type이 EMBEDDING이거나 STORAGE일 때, BINDATASTORAGE에 저장된 바이너리 데이터의 아이디
WORD 2 - - Type이 EMBEDDING일 때, 바이너리 데이터의 형식 이름의 길이 (len3)
WCHAR array[len3] 2 x len3 String? extensionForEmbedding Type이 EMBEDDING일 때, extension(. 제외)[1][2]
전체 길이 가변 10 + (2xlen1) + (2xlen2) + (2xlen3)

# 속성

바이너리 데이터의 속성을 나타냅니다.

libhwp에서 HWPBinDataProperty객체로 표현됩니다.

범위 구분 타입 설명
bit 0-3 Type HWPBinDataType 종류
bit 4-5 Compress HWPBinDataCompress 압축
bit 8-9 State HWPBinDataCompress 상태
전체 길이 2

# 종류

바이너리 데이터의 종류를 나타냅니다.

libhwp에서 enum class인 HWPBinDataType객체로 표현됩니다.

값 상수 설명
0x0000 Link 그림 외부 파일 참조
0x0001 Embedding 그림 파일 포함
0x0002 Storage OLE 포함

# 압축

바이너리 데이터의 압축을 나타냅니다.

libhwp에서 enum class인 HWPBinDataCompress객체로 표현됩니다.

값 상수 설명
0x0000 ByStorageDefault 스토리지의 디폴트 모드 따라감
0x0010 Compress 무조건 압축
0x0020 NoCompress 무조건 압축 하지 않음

# 상태

바이너리 데이터의 상태를 나타냅니다.

libhwp에서 enum class인 HWPBinDataState객체로 표현됩니다.

값 상수 설명
0x0000 NotAccess 아직 access된 적이 없는 상태
0x0100 SuccessAccess access에 성공하여 파일을 찾은 상태
0x0200 FailAccess access가 실패한 에러 상태
0x0300 FailAccessButIgnore 링크 access가 실패했으나 무시된 상태
