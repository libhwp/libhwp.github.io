# 스타일

Tag ID

HWPTAG_STYLE

# 데이터 표

스타일을 나타냅니다.

libhwp에서 HWPStyle객체로 표현됩니다.

자료형 길이 타입 변수 설명
WORD 2 - - 길이 (len1)
WCHAR array[len1] 2xlen1 String? hangulName 로컬 스타일 이름 [1]
WORD 2 - - 길이 (len2)
WCHAR array[len2] 2xlen2 String? englishName 영문 스타일 이름
BYTE 1 HWPStyleProperty property 속성
BYTE 1 Short nextStyleId 다음 스타일 아이디
INT16 2 Short languageId 언어 아이디
UINT16 2 Int paraShapeId 문단 모양 아이디 [2]
UINT16 2 Int charShapeId 글자 모양 아이디 [3]
전체 길이 가변 12 + (2xlen1) + (2xlen2)

# 속성

속성을 나타냅니다.

libhwp에서 HWPParaShapeProperty1객체로 표현됩니다.

범위 타입 변수 설명
bit 0-2 HWPStyleSort _styleSort 스타일 종류
전체 길이 1

# 스타일 종류

스타일 종류를 나타냅니다.

libhwp에서 enum class인 HWPStyleSort객체로 표현됩니다.

값 상수 설명
0 ParaStyle 문단 스타일
1 CharStyle 글자 스타일

  1. 한글 윈도우에서는 한글 스타일 이름 ↩︎

  2. 스타일의 종류가 문단인 경우 반드시 지정해야 한다. ↩︎

  3. 스타일의 종류가 글자인 경우 반드시 지정해야 한다. ↩︎
