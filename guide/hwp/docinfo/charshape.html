# 글자 모양

Tag ID

HWPTAG_CHAR_SHAPE

# 데이터 표

글자 모양을 나타냅니다.

libhwp에서 HWPCharShape객체로 표현됩니다.

자료형 길이 타입 변수 설명
WORD array[7] 14 HWPFaceNameIds faceNameIds 언어별 글꼴 ID 참조 값
UINT8 array[7] 7 HWPRatios ratios 언어별 장평
(50% - 200%)
INT8 array[7] 7 HWPCharSpaces charSpaces 언어별 자간
(-50% - 50%)
UINT8 array[7] 7 HWPRelativeSizes relativeSizes 언어별 상대 크기
(10% - 250%)
INT8 array[7] 7 HWPCharOffsets charOffsets 언어별 글자 위치
(-100% - 100%)
INT32 4 Int baseSize 기준 크기
(0pt - 4096pt)
UINT32 4 HWPCharShapeProperty property 속성
INT8 1 Byte shadowGap1 그림자 간격
(-100% - 100%)
INT8 1 Byte shadowGap2 그림자 간격
(-100% - 100%)
COLORREF 4 Color4Byte charColor 글자 색
COLORREF 4 Color4Byte underLineColor 밑줄 색
COLORREF 4 Color4Byte shadeColor 음영 색
COLORREF 4 Color4Byte shadowColor 그림자 색
UINT16 2 Int borderFillId 글자 테두리/배경 ID
(5.0.2.1 이상)
COLORREF 4 Color4Byte strikeLineColor 취소선 색
(5.0.3.0 이상)
전체 길이 72

# 속성

글자 모양 속성을 나타냅니다.

libhwp에서 HWPCharShapeProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 Boolean _italic 기울임 여부
bit 1 Boolean _bold 진하게 여부
bit 2-3 HWPUnderLineSort _underLineSort 밑줄 종류
bit 4-7 HWPBorderType _underLineShape 밑줄 모양
bit 8-10 HWPOuterLineSort _outerLineSort 외곽선 종류
bit 11-12 HWPShadowSort _shadowSort 그림자 종류
bit 13 Boolean _emboss 양각 여부
bit 14 Boolean _engrave 음각 여부
bit 15 Boolean _superScript 위 첨자 여부
bit 16 Boolean _subScript 아래 첨자 여부
bit 18-20 Boolean _strikeLine 취소선 여부
bit 21-24 HWPEmphasisSort _emphasisSort 강조점 종류
bit 25 Boolean _usingSpaceAppropriateForFont 글꼴에 어울리는 빈칸 사용 여부
bit 26-29 HWPBorderType _strikeLineShape 취소선 모양
bit 30 Boolean _kerning Kerning 여부
전체 길이 4

# 글꼴에 대한 언어

글꼴에 대한 언어를 나타냅니다.

libhwp에서 enum class인 HWPFaceNameEnum객체로 표현됩니다.

값 상수 설명
0 HANGUL 한글
1 ENGLISH 영어
2 HANJA 한자
3 JAPANESE 일본어
4 ETC 기타
5 SYMBOL 기호
6 USER 사용자

# 밑줄 종류

밑줄 종류를 나타냅니다.

libhwp에서 enum class인 HWPUnderLineSort객체로 표현됩니다.

값 상수 설명
0 None 없음
1 Bottom 글자 아래
3 Top 글자 위

# 외각선 종류

외각선 종류를 나타냅니다.

libhwp에서 enum class인 HWPOuterLineSort객체로 표현됩니다.

값 상수 설명
0 None 없음
1 Solid 실선
2 Dot 점선
3 BoldSolid 굵은 실선(두꺼운 선)
4 Dash 파선(긴 점선)
5 DashDot 일점쇄선(-.-.-.-.-.)
6 DashDotDot 이점쇄선(-..-..-..-..)

# 그림자 종류

그림자 종류를 나타냅니다.

libhwp에서 enum class인 HWPShadowSort객체로 표현됩니다.

값 상수 설명
0 None 없음
1 Discontinuous 비연속
2 Continuous 연속

# 강조점 종류

강조점 종류를 나타냅니다.

libhwp에서 enum class인 HWPEmphasisSort객체로 표현됩니다.

값 상수 설명
0 None 없음
1 Circle 검정 동그라미 강조점
2 EmptyCircle 속 빈 동그라미 강조점
3 Type3 ˇ
4 Type4 ~
5 Type5 .
6 Type6 :
