# 문단 모양

Tag ID

HWPTAG_PARA_SHAPE

# 데이터 표

문단 모양을 나타냅니다.

libhwp에서 HWPParaShape객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 HWPParaShapeProperty1 property1 속성1
INT32 4 Int leftMargin 왼쪽 여백
INT32 4 Int rightMargin 오른쪽 여백
INT32 4 Int indent 들여 쓰기/내어 쓰기
INT32 4 Int topParaSpace 문단 간격 위
INT32 4 Int BottomParaSpace 문단 간격 아래
INT32 4 Int lineSpace 줄 간격 [1]
UINT16 2 Int tabDefId 탭 정의 아이디
UINT16 2 Int paraHeadId 번호 문단 / 글머리표 아이디
UINT16 2 Int borderFillId 테두리/배경 아이디
INT16 2 Short leftBorderSpace 문단 테두리 왼쪽 간격
INT16 2 Short rightBorderSpace 문단 테두리 오른쪽 간격
INT16 2 Short topBorderSpace 문단 테두리 위쪽 간격
INT16 2 Short bottomBorderSpace 문단 테두리 아래쪽 간격
UINT32 4 HWPParaShapeProperty2 property1 속성2 [2]
UINT32 4 HWPParaShapeProperty3 property1 속성3 [3]
UINT32 4 Long property1 줄 간격 [3:1]

# 문단 모양 속성1

문단 모양 속성1을 나타냅니다.

libhwp에서 HWPParaShapeProperty1객체로 표현됩니다.

범위 타입 변수 설명
bit 0-1 HWPLineSpaceSort _lineSpaceSort 줄 간격 종류
bit 2-4 HWPAlignment _aligment 정렬 방식
bit 5-6 HWPLineDivideForEnglish _lineDivideForEnglish 줄 나눔 기준 영어 단위
bit 7 HWPLineDivideForHangul _lineDivideForHangul 줄 나눔 기준 한글 단위
bit 8 Boolean _useGrid 편집 용지의 줄 격자 사용 여부
bit 9-15 Byte _minimumSpace 공백 최소값 [4]
bit 16 Boolean _protectLonger 외톨이줄 보호 여부
bit 17 Boolean _togetherNextPara 다음 문단과 함께 여부
bit 18 Boolean protectPara 문단 보호 여부
bit 19 Boolean _splitPageBeforePara 문단 앞에서 항상 쪽 나눔 여부
bit 20-21 HWPVerticalAlignment _verticalAlignment 세로 정렬
bit 22 Boolean _lineHeightForFont 글꼴에 어울리는 줄 높이 여부
bit 23-24 HWPParaHeadShape _paraHeadShape 문단 머리 모양 종류
bit 25-27 Byte _paralevel 문단 수준 [5]
bit 28 Boolean _linkBorder 문단 테두리 연결 여부
bit 29 Boolean _ignoreParaMargin 문단 여백 무시 여부
bit 30 Boolean _paraTailShape 문단 꼬리 모양

# 문단 모양 속성2

문단 모양 속성2을 나타냅니다.

libhwp에서 HWPParaShapeProperty2객체로 표현됩니다.

범위 타입 변수 설명
bit 0-1 Boolean _inputSingleLine 한줄로 입력 여부
bit 4 Boolean _autoAdjustGapHangulEnglish 한글과 영어 간격 자동 조절 여부
bit 5 Boolean _autoAdjustHangulNumber 한글과 숫자 간격 자동 조절 여부

# 문단 모양 속성3

문단 모양 속성3을 나타냅니다.

libhwp에서 HWPParaShapeProperty3객체로 표현됩니다.

범위 타입 변수 설명
bit 0-4 HWPLineSpaceSort _lineSpaceSort 줄 간격 종류

# 줄 간격 종류

WARNING

한글 2007이하 버전 (5.0.2.5 버전 미만)에서 사용.

property3 에서는 사용 가능합니다.

줄 간격 종류를 나타냅니다.

libhwp에서 enum class인 HWPLineSpaceSort객체로 표현됩니다.

값 상수 설명
0 RatioForLetter 글자에 따라(%)
1 FixedValue 고정값
2 OnlyMargin 여백만 지적
3 Minimum 최소

# 정렬 종류

정렬 종류를 나타냅니다.

libhwp에서 enum class인 HWPAlignment객체로 표현됩니다.

값 상수 설명
0 Justify 양쪽 정렬
1 Left 왼쪽 정렬
2 Right 오른쪽 정렬
3 Center 가운데 정렬
4 Distribute 배분 정렬
5 Divide 나눔 정렬

# 줄 나눔 기준 영어 단위

줄 나눔 기준 영어 단위를 나타냅니다.

libhwp에서 enum class인 HWPLineDivideForEnglish객체로 표현됩니다.

값 상수 설명
0 ByWord 단어
1 ByHyphen 하이픈
2 ByLetter 글자

# 줄 나눔 기준 한글 단위

줄 나눔 기준 한글 단위를 나타냅니다.

libhwp에서 enum class인 HWPLineDivideForHangul객체로 표현됩니다.

값 상수 설명
0 ByWord 어절
1 ByLetter 글자

# 세로 정렬

세로 정렬를 나타냅니다.

libhwp에서 enum class인 HWPVerticalAlignment객체로 표현됩니다.

값 상수 설명
0 ByFont 글꼴 기준 정렬
1 Top 위쪽 정렬
2 Center 가운데 정렬
3 Bottom 아래 정렬

# 문단 머리 모양 종류

문단 머리 모양 종류를 나타냅니다.

libhwp에서 enum class인 HWPParaHeadShape객체로 표현됩니다.

값 상수 설명
0 None 없음
1 Outline 개요
2 Numbering 번호
3 Bullet 글머리표(bullet)

  3. 5.0.2.5 버전 이상 ↩︎ ↩︎

  4. 0%-75% ↩︎
