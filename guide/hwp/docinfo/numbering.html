# 문단 번호

Tag ID

HWPTAG_NUMBERING

# 데이터 표

문단 번호을 나타냅니다.

libhwp에서 HWPNumbering객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream 가변 HWPLevelNumbering levelNumbering 번호 형식 [1] [2]
UINT16 2 Int startNumber 시작 번호
UINT 4 Long startNumberForLevel 수준별 시작번호 [3]
BYTE stream 가변 HWPExtendNumbering extendNumbering 확장 번호 형식 new [4] [2:1]
UINT 4 Long startNumberForExtend 확장 수준별 시작번호 new [5] [4:1]
전체 길이 가변 (12 + (2+len)) * 7 + 2 + ((4x7) + (2xlen)) x 3 + (4 x 3)

# 번호 형식

번호 형식을 나타냅니다.

libhwp에서 HWPLevelNumbering객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream 12 HWPParagraphHeadInfo paragraphHeadInfo 문단 머리 정보
WORD 2 - - 번호 형식 길이 (len)
WCHAR array[len] 2 x len String? numberFormat 번호 형식 [6]
전체 길이 가변 12 + (2 x len)

# 확장 번호 형식 new

확장 번호 형식을 나타냅니다.

libhwp에서 HWPExtendNumbering객체로 표현됩니다.

자료형 길이 타입 변수 설명
WORD 2 - - 번호 형식 길이 (len)
WCHAR array[len] 2 x len String? numberFormat 확장 번호 형식 [6:1]
전체 길이 가변 2 x len

# 문단 머리 정보

문단 머리 정보를 나타냅니다.

libhwp에서 HWPParagraphHeadInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT 4 PROPERTY[7] property 속성
HWPUNIT16 2 Short correctionValueForWidth 너비 보정값
HWPUNIT16 2 Short property 본문과의 거리
UINT 4 Long property 글자 모양 아이디 참조
전체 길이 12

# 문단 머리 정보 속성

문단 머리 정보 속성을 나타냅니다.

libhwp에서 HWPParagraphHeadInfoProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0-1 HWPParagraphAlignment _paragraphAlignment 문단의 정렬 종류
bit 2 Boolean _followStringWidth 번호 너비를 실제 인스턴스 문자열의 너비에 따를지 여부
bit 3 Boolean _autoIndent 자동 내어 쓰기 여부
bit 4 HWPValueType _valueTypeForDistanceFromBody 수준별 본문과의 거리 종류
전체 길이 4

# 문단의 정렬 종류

문단의 정렬 종류를 나타냅니다.

libhwp에서 enum class인 HWPParagraphAlignment객체로 표현됩니다.

값 상수 설명
0 Left 왼쪽
1 Center 가운데
2 Right 오른쪽

# 수준별 본문과의 거리 종류

수준별 본문과의 거리 종류를 나타냅니다.

libhwp에서 enum class인 HWPValueType객체로 표현됩니다.

값 상수 설명
0 RatioForLetter 글자 크기에 대한 상대 비율
1 Value 값

# 문단 번호 형식

문단 번호 형식을 나타냅니다.

libhwp에서 enum class인 HWPNumberShape객체로 표현됩니다.

값 상수 설명
0 Type0 1, 2, 3
1 Type1 동그라미 쳐진 1, 2, 3
2 Type2 I, II, III
3 Type3 i, ii, iii
4 Type4 A, B, C
5 Type5 a, c, c
6 Type6 동그라미 쳐진 A, B, C
7 Type7 동그라미 쳐진 a, b, c
8 Type8 가, 나, 다
9 Type9 동그라미 쳐진 가, 나, 다
10 Type10 ㄱ, ㄴ, ㄷ
11 Type11 동그라미 쳐진 ㄱ, ㄴ, ㄷ
12 Type12 일, 이, 삼
13 Type13 一, 二, 三
14 Type14 동그라미 쳐진 一, 二, 三
15 Type15 갑, 을, 병, 정, 무, 기, 경, 신, 임, 계
16 Type16 甲, 乙, 丙, 丁, 戊, 己 庚, 辛, 壬, 癸
0x80 Type128 4가지 문자가 차례로 반복 [8]
0x81 Type129 사용자 지정 문자 반복 [8:1]

  1. 7회 반복, 수준 (1 ~ 7) ↩︎

  2. 각 레벨에 해당하는 숫자 또는 문자 또는 기호를 표시 ↩︎ ↩︎

  3. 5.0.2.5 이상 ↩︎

  4. 3 회 반복, 수준 (8-10) ↩︎ ↩︎

  5. 5.1.0.0 이상 ↩︎

  6. 불릿 문단의 경우에는 사용되지 않는다. 문자열 내 특정 문자에 제어코드 (^)를 붙힘으로써 한글에서 표시되는 번호 문단 머리의 형식을 제어한다. ^n: 레벨 경로를 표시한다. (예: 1.1.1.1.1.1.1), ^N: 레벨 경로를 표시하며 마지막에 마침표를 하나 더 찍는다. (예: 1.1.1.1.1.1.1.) ↩︎ ↩︎

  7. HWPParagraphHeadInfoProperty ↩︎
