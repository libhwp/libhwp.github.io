# 탭 정의

Tag ID

HWPTAG_TAB_DEF

# 데이터 표

탭 정의을 나타냅니다.

libhwp에서 HWPTabDef객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 HWPTabDefProperty property 속성
INT32 4 - - count
BYTE stream 8 x count HWPTabInfo tabInfo 탭 정보
(count 개수)
전체 길이 8 + (8 x count)

# 속성

탭 정의 속성을 나타냅니다.

libhwp에서 HWPTabDefProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 Boolean _autoTabAtParagraphLeftEnd 문단 왼쪽 끝 자동 탭 유무[1]
bit 1 Boolean _autoTabAtParagraphRightEnd 문단 오른쪽 끝 자동 탭 유무
전체 길이 4

# 탭 정보

탭 정보를 나타냅니다.

libhwp에서 HWPTabInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
HWPUNIT 4 Long position 탭의 위치
UINT8 1 HWPTabSort tabSort 탭의 종류
UINT8 1 HWPBorderType fillSort 채움 종류
UINT16 2 - - 8 바이트를 맞추기 위한 예약
전체 길이 8

# 탭 종류

탭 종류를 나타냅니다.

libhwp에서 enum class인 HWPTabSort객체로 표현됩니다.

값 상수 설명
0 Left 왼쪽
1 Right 오른쪽
2 Center 가운데
3 DecimalPoint 소수점
