# 테두리/배경

Tag ID

HWPTAG_BORDER_FILL

# 데이터 표

테두리/배경을 나타냅니다.

libhwp에서 HWPBorderFill객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT16 2 HWPBorderFillProperty property 속성
BYTE stream 6 HWPEachBorder leftBorder 왼쪽 테두리선
BYTE stream 6 HWPEachBorder rightBorder 오른쪽 테두리선
BYTE stream 6 HWPEachBorder topBorder 위쪽 테두리선
BYTE stream 6 HWPEachBorder bottomBorder 아래쪽 테두리선
UINT8 1 HWPDiagonalSort diagonalSort 대각선 종류
UINT8 1 HWPBorderThickness diagonalThickness 대각선 굵기
COLORREF 4 Color4Byte diagonalColor 대각선 색깔
BYTE stream n HWPFillInfo fillInfo 채우기 정보
전체 길이 가변 32 + n

# 속성

테두리/배경 속성을 나타냅니다.

libhwp에서 HWPBorderFillProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 Boolean _3DEffect 3D 효과의 유무
bit 1 Boolean _shadowEffect 그림자 효과의 유무
bit 2-4 HWPSlashDiagonalShape _slashDiagonalShape Slash 대각선 모양
(시계 방향으로 각각의 대각선 유무를 나타냄)
bit 5-7 HWPBackSlashDiagonalShape _backSlashDiagonalShape BackSlash 대각선 모양
(반시계 방향으로 각각의 대각선 유무를 나타냄)
bit 8-9 Boolean _brokenSlashDiagonal Slash 대각선 꺽은선
bit 10 Boolean _brokenBackSlashDiagonal BackSlash 대각선 꺽선
bit 11 Boolean _rotateSlashDiagonal180 Slash 대각선 모양 180도 회전 여부
bit 12 Boolean _rotateBackSlashDiagonal180 BackSlash 대각선 모양 180도 회전 여부
bit 13 Boolean _centerLine 중심선 유무
전체 길이 2

# 테두리선

테두리선을 나타냅니다.

libhwp에서 HWPEachBorder객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT8 1 HWPBorderType type 테두리선 종류
UINT8 1 HWPBorderThickness thickness 테두리선 굵기
COLORREF 4 Color4Byte color 테두리선 색상
전체 길이 6

# 테두리선 종류

테두리선 종류를 나타냅니다.

libhwp에서 enum class인 HWPBorderType객체로 표현됩니다.

값 상수 설명
0 Solid 실선
1 Dash 긴 점선
2 Dot 점선
3 DashDot -.-.-.-.
4 DashDotDot -..-..-..
5 LongDash Dash 보다 긴 선분의 반복
6 CircleDot Dot 보다 큰 동그라미의 반복
7 Double 2중선
8 ThickThick 가는선 + 굵은선 2중선
9 ThickThin 굵은선 + 가는선 2중선
10 ThinThickThin 가는선 + 굵은선 + 가는선 3중선
11 Wave 물결
12 DoubleWave 물결 2중선
13 Thick3D 두꺼운 3D
14 Thick3DReverseLighting 두꺼운 3D(광원 반대)
15 Solid3D 3D 단선
16 Solid3DReverseLighting 3D 단선(광원 반대)

libhwp에서 enum class인 HWPBorderThickness객체로 표현됩니다.

8 MM0_6 0.6 mm
9 MM0_7 0.7 mm

14 MM4_0 4.0 mm

# 대각선 종류

대각선 종류를 나타냅니다.

libhwp에서 enum class인 HWPDiagonalSort객체로 표현됩니다.

값 상수 설명
0 Slash Slash
1 BackSlash BackSlash
2 CrookedSlash CrookedSlash

# Slash 대각선 모양

Slash 대각선 모양을 나타냅니다.

libhwp에서 enum class인 HWPSlashDiagonalShape객체로 표현됩니다.

값 상수 설명
000 None None
010 Slash slash
011 LeftTopToBottomEdge Left Top --> Bottom Edge
110 LeftTopToRightEdge Left Top --> Right Edge
111 LeftToBottomRightEdge Left Top --> Bottom & Right Edge

# BackSlash 대각선 모양

BackSlash 대각선 모양을 나타냅니다.

libhwp에서 enum class인 HWPBackSlashDiagonalShape객체로 표현됩니다.

값 상수 설명
000 None None
010 BackSlash / back slash
011 RightTopToBottomEdge Right Top --> Bottom Edge
110 RightTopToKLeftEdge Right Top --> Left Edge
111 RightTopToBottomLeftEdge Right Top --> Bottom & Left Edge

# 채우기 정보

채우기 정보를 나타냅니다.

libhwp에서 HWPFillInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT 4 HWPFillType type 채우기 종류(type)
BYTE stream 12 HWPPatternFill? patternFill 단색 채우기
BYTE stream 가변 HWPGradientFill? gradientFill 그러데이션 채우기
BYTE stream 가변 HWPImageFill? imageFill 이미지 채우기
전체 길이 가변 8 + n

# 채우기 종류

채우기 종류를 나타냅니다.

libhwp에서 HWPFillType객체로 표현됩니다.

범위 타입 변수 설명
0x00000000 Boolean - 채우기 없음
0x00000001 Boolean _patternFill 단색 채우기 유무
0x00000002 Boolean _imageFill 이미지 채우기 유무
0x00000004 Boolean _gradientFill 그러데이션 채우기 유무

# 단색 채우기

존재 유무

type & 0x0000001 != 0

단색 채우기를 나타냅니다.

libhwp에서 HWPPatternFill객체로 표현됩니다.

자료형 길이 타입 변수 설명
COLORREF 4 Color4Byte backColor 배경색
COLORREF 4 Color4Byte patternColor 무늬색
INT32 4 HWPPatternType patternType 무늬 종류
전체 길이 12

# 그러데이션 채우기

존재 유무

type & 0x0000004 != 0

그러데이션 채우기를 나타냅니다.

libhwp에서 HWPGradientFill객체로 표현됩니다.

자료형 길이 타입 변수 설명
INT8 1 HWPGradientType gradientType 그러데이션 유형
UINT32 4 Long startAngle 그러데이션 기울임
(시작각)
UINT32 4 Long centerX 그러데이션 가로 중심
(중심 X 좌표)
UINT32 4 Long centerY 그러데이션 세로 중심
(중심 Y 좌표)
UINT32 4 Long blurringDegree 그러데이션 번짐 정도
(0-100)
INT16 2 - - 그러데이션 색 수(num)[1]
INT32 4 x num ArrayList<Int> changePointList 색상이 바뀌는 곳의 위치
(num > 2)
COLORREF array[num] 4 x num ArrayList<Color4Byte> colorList 색상
BYTE 1 Short blurringCenter 번짐 정도의 중심
(0-100)[2]
전체 길이 가변 20 + (4 x num) + (4 x num)

# 이미지 채우기

존재 유무

type & 0x0000002 != 0

이미지 채우기를 나타냅니다.

libhwp에서 HWPImageFill객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE 1 HWPImageFillType imageFillType 이미지 채우기 유형
BYTE stream 5 HWPPictureInfo pictureInfo 그림 정보
DWORD 4 - - size[3]
DWORD 4 - - 추가 채우기 속성 길이 (size)
전체 길이 가변 6 + size

# 채우기 무늬 종류

채우기 무늬 종류를 나타냅니다.

libhwp에서 enum class인 HWPPatternType객체로 표현됩니다.

값 상수 설명
-1 None None
1 HorizontalLine ----
2 VerticalLine |||||
3 BackDiagonalLine \\\\\
4 FrontDiagonalLine /////
5 CrossLine +++++
6 CrossDiagonalLine xxxxx

# 그러데이션 유형

그러데이션 무늬 종류를 나타냅니다.

libhwp에서 enum class인 HWPGradientType객체로 표현됩니다.

값 상수 설명
1 Stripe 줄무늬형
2 Circle 원형
3 Cone 원뿔형
4 Square 사각형

# 이미지 채우기 유형

이미지 채우기 유형을 나타냅니다.

libhwp에서 enum class인 HWPImageFillType객체로 표현됩니다.

값 상수 설명
0 TileAll 바둑판식으로 - 모두
1 TileHorizontalTop 바둑판식으로 - 가로/위
2 TileHorizontalBottom 바둑판식으로 - 가로/아래
3 TileVerticalLeft 바둑판식으로 - 세로/왼쪽
4 TileVerticalRight 바둑판식으로 - 세로/오른쪽
5 FitSize 크기에 맞추어
6 Center 가운데로
7 CenterTop 가운데 위로
8 CenterBottom 가운데 아래로
9 LeftCenter 왼쪽 가운데로
10 LeftTop 왼쪽 위로
11 LeftBottom 왼쪽 아래로
12 RightCenter 오른쪽 가운데로
13 RightTop 오른쪽 위로
14 RightBottom 오른쪽 아래로
15 None NONE

# 그림 정보

그림 정보를 나타냅니다.

libhwp에서 HWPPictureInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
INT8 1 Byte brightness 밝기
INT8 1 Byte contrast 명암
BYTE 1 HWPPictureEffect effect 그림 효과
UINT16 2 binItemID binItemID BinData의 아이디 참조값
전체 길이 5

# 그림 효과

그림 효과를 나타냅니다.

libhwp에서 enum class인 HWPPictureEffect객체로 표현됩니다.

값 상수 설명
0 RealPicture REAL_PIC
1 GrayScale GRAY_SCALE
2 BalckWhite BLACK_WHITE
4 Pattern8x8 PATTERN8x8

  1. 워디안/한글2002/SE에서는 항상 2이다. ↩︎

  2. 추가 채우기 속성, 그러데이션일 경우 ↩︎

  3. type의 bit 0x004가 켜져 있으면 1, 아니면 0 ↩︎
