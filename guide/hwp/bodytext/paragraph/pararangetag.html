# 문단의 영역 태그

Tag ID

HWPTAG_PARA_RANGE_TAG

TIP

range tag정보를 정보 수만큼 읽어 온다. range tag는 텍스트의 일정 영역을 마킹하는 용도로 사용되며, 글자 모양과는 달리 각 영역은 서로 겹칠 수 있다. (형광펜, 교정 부호, ...)

# 데이터 표

문단의 영역 태그를 나타냅니다.

libhwp에서 HWPParaRangeTag객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream 12 HWPRangeTagItem rangeTagItem 영역 태그 아이탬
전체 길이 가변 12 x n

# 영역 태그 아이템

영역 태그 아이템를 나타냅니다.

libhwp에서 HWPRangeTagItem객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 Long rangeStart 영역 시작
UINT32 4 Long rangeEnd 영역 끝
UINT32 4 Short|ByteArray? sort|data 태그(종류 + 데이터) [1]

  1. 상위 8비트가 종류를, 하위 24비트가 종류별로 다른 설명을 부여할 수 있는 임의의 데이터를 나타냅니다. ↩︎
