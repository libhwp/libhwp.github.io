# 문단 헤더

Tag ID

HWPTAG_PARA_HEADER

# 데이터 표

문단 헤더를 나타냅니다.

libhwp에서 HWPParaHeadear객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 Long characterCount text(=chars)[1]
UINT32 4 HWPControlMask controlMask control mask [2]
UINT16 2 Int paraShapeId 문단 모양 아이디
UINT8 1 Short styleId 문단 스타일 아이디
UINT8 1 HWPDivideSort divideSort 단 나누기 종류
UINT16 2 Int charShapeCount 글자 모양 정보 수
UINT16 2 Int rangeTagCount range tag 정보 수
UINT16 2 Int lineAlignCount 각 줄의 align에 대한 정보 수
UINT32 4 Long instanceID 문단 Instance ID (unique ID)
UINT16 2 Int isMergedByTrack 변경추적 병합 문단 여부 [3]

# Control Mask

Control Mask을 나타냅니다.

libhwp에서 HWPControlMask객체로 표현됩니다.

범위 타입 변수 설명
bit 2 Boolean _hasSectColDef 구역/단 정의
bit 3 Boolean _hasFieldStart 필드 시작
bit 4 Boolean _hasFieldEnd 필드 끝
bit 8 Boolean _hasTitleMark Title Mark
bit 9 Boolean _hasTab 탭
bit 10 Boolean _hasLineBreak 강제 줄 나눔
bit 11 Boolean _hasGsoTable 그리기 개체 / 표
bit 13 Boolean _hasParaBreak 문단 나누기
bit 15 Boolean _hasHiddenComment 주석
bit 16 Boolean _hasHeaderFooter 머리말 / 꼬리말
bit 17 Boolean _hasFootnoteEndnote 각주 / 미주
bit 18 Boolean _hasAutoNumber 자동 번호
bit 21 Boolean _hasPageControl 쪽 바뀜
bit 22 Boolean _hasBookmark 책갈피 / 찾아보기
bit 23 Boolean _hasAdditionalTextOverlappingLetter 덧말 / 글자 겹침
bit 24 Boolean _hasHyphen 하이픈
bit 30 Boolean _hasBundleBlank 묶음 빈칸
bit 31 Boolean _hasFixWidthBlank 고정 폭 빈칸

# 단 나누기 종류

단 나누기 종류를 나타냅니다.

libhwp에서 HWPDivideSort객체로 표현됩니다.

범위 타입 변수 설명
0x01 Boolean _isDivideMultiColumn 구역 나누기
0x02 Boolean _isDivideColumn 다단 나누기
0x04 Boolean _isDividePage 쪽 나누기
0x08 Boolean _isDivideSection 단 나누기

  1. if(nchars & 0x80000000) nchars &= 0x7fffffff, 첫 비트로 문단의 마지막 여부를 알 수 있다. ↩︎

  2. (UINT32)(1 << ctrlch) 조합 ↩︎

  3. 5.0.3.2 버전 이상 ↩︎
