# 쪽 번호 위치

Control ID

makeCtrlID('p', 'g, 'n', 'p')

# 데이터 표

쪽 번호 위치를 나타냅니다.

libhwp에서 HWPControlPageNumberPosition객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream 18 ...[1] header 쪽 번호 위치 헤더
BYTE stream n HWPCtrlData? ctrlData 컨트롤 데이터
전체 길이 가변 18 + n

# 쪽 번호 위치 헤더

쪽 번호 위치 헤더를 나타냅니다.

libhwp에서 HWPCtrlHeaderPageNumberPosition객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 Long ctrlId 컨트롤 ID
UINT32 4 ...[2] property 속성
UINT16 2 Int number 번호
WCHAR 2 String? userSymbol 사용자 기호
WCHAR 2 String? beforeDecorationLetter 앞 장식 문자
WCHAR 2 String? afterDecorationLetter 뒤 장식 문자
WCHAR 2 String? - 항상 "-"
전체 길이 18

# 쪽 번호 위치 헤더 속성

쪽 번호 위치 헤더 속성을 나타냅니다.

libhwp에서 PageNumberPositionHeaderProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0-7 HWPNumberShape _numberShape 번호 모양
bit 8-11 HWPNumberPosition _numberPosition 번호의 위치 표시

# 번호의 위치 표시

번호의 위치 표를 나타냅니다.

libhwp에서 enum class인 HWPNumberSort객체로 표현됩니다.

값 상수 설명
0 None 쪽 번호 없음
1 LeftTop 왼쪽 위
2 CenterTop 가운데 위
3 RightTop 오른쪽 위
4 LeftBottom 왼쪽 아래
5 CenterBottom 가운데 아래
6 RightBottom 오른쪽 아래
7 OutsideTop 바깥쪽 위
8 OutsideBottom 바깥쪽 아래
9 InsideTop 안쪽 위
10 InsideBottom 안쪽 아래
