# 그리기 개체

그리기 개체

선, 사각형, 타원, 호, 다각형, 곡선

TIP

모든 그리기 개체에 대한 serialization은 우선 base인 그리기 개체 공통 속성을 serialize한 다음 자신이 가지고 있는 개체 요소 속성을 serialize합니다.

그리기 개체 요소 공통 속성을 나타냅니다.

libhwp에서 HWPShapeComponentNormal객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream n HWPShapeComponent shapeComponent 개체 요소 속성
BYTE stream 11 HWPLineInfo? lineInfo 테두리 선 정보
BYTE stream n2 HWPFillInfo? fillInfo 채우기 정보
BYTE stream n3 HWPShadowInfo? shadowInfo 그림자 정보
전체 길이 가변 n + 11 + n2 + n3

# 개체 요소

Tag ID

HWPTAG_SHAPE_COMPONENT

GenShapeObject일 경우 id가 두 번 기록됩니다.

개체 요소 속성을 나타냅니다.

libhwp에서 HWPShapeComponent객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 Long? gsoId 개체 컨트롤 ID
INT32 4 Int offsetX 그룹 내 X offset
INT32 4 Int offsetY 그룹 내 Y offset
WORD 2 Int groupingCount 몇 번이나 그룹
WORD 2 Int localFileVersion local file version
UINT32 4 Long widthAtCreate 개체 생성시 초기 폭
UINT32 4 Long heightAtCreate 개체 생성시 초기 높이
UINT32 4 Long widthAtCurrent 개체 현재 폭
UINT32 4 Long heightAtCurrent 개체 현재 높이
UINT32 4 Long property 속성
horz flip(0) / vert flip(1)
HWPUNIT16 2 Int rotateAngle 회전 각
INT32 4 Int rotateXCenter 회전 중심 x 좌표
INT32 4 Int rotateYCenter 회전 중심 y 좌표
BYTE stream n HWPRenderingInfo renderingInfo Rendering 정보
전체 길이 가변 42 + n

# Rendering 정보

Rendering 정보을 나타냅니다.

libhwp에서 HWPRenderingInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
WORD 2 - - scale/rotation matrix 쌍의 개수 (cnt) [1]
BYTE stream 48 HWPMatrix translationMatrix translation matrix
BYTE stream cntx48x2 HWPMatrix scaleRoatateMatrix scale/rotation matrix sequence
전체 길이 가변 50 + (cnt x 48 x 2)

# Scale/Rotation Matrix 정보

Scale/Rotation Matrix 정보을 나타냅니다.

libhwp에서 HWPScaleRotateMatrixPair객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream 48 HWPMatrix scaleMatrix scale matrix
BYTE stream 48 HWPMatrix rotateMatrix rotate matrix
전체 길이 96

# Matrix 정보

Matrix 정보을 나타냅니다.

libhwp에서 HWPMatrix객체로 표현됩니다.

자료형 길이 타입 변수 설명
Double 8 Double values values[0]
Double 8 Double values values[1]
Double 8 Double values values[2]
Double 8 Double values values[3]
Double 8 Double values values[4]
Double 8 Double values values[5]
전체 길이 48

TIP

각 matrix 원소가 double로 표현되는 3 x 3 matrix로 구현됩니다. 마지막 줄(row)은 항상 0, 0, 1이기 때문에 실제 serialization에서는 마지막 줄이 빠집니다.

# 테두리 선 정보

테두리 선 정보을 나타냅니다.

libhwp에서 HWPLineInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
COLORREF 4 Color4Byte color 선 색상
INT16 2 Int thickness 선 굵기
UINT32 4 HWPLineInfoProperty property 속성
BYTE 1 HWPOutlineStyle outlineStyle Outline Style
전체 길이 11

# 테두리 선 정보 속성

테두리 선 정보 속성을 나타냅니다.

libhwp에서 HWPLineInfoProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0-5 HWPLineType _lineType 선 종류
bit 6-9 HWPLineEndShape _lineEndShape 선 끝 모양
bit 10-15 HWPLineArrowShape _startArrowShape 화살표 시작 모양
bit 16-21 HWPLineArrowShape _endArrowShape 화살표 끝 모양
bit 22-25 HWPLineArrowSize _startArrowSize 화살표 시작 크기
bit 26-29 HWPLineArrowSize _endArrowSize 화살표 끝 크기
bit 30 Boolean _isFillStartArrow 시작부분 화살표 채움 여부
bit 31 Boolean _isFillEndArrow 끝부분 화살표 채움 여부

# 선 종류

선 종류를 나타냅니다.

libhwp에서 enum class인 HWPLineType객체로 표현됩니다.

값 상수 설명
0 None 없음
1 Solid 실선
2 Dash 긴 점선
3 Dot 점선
4 DashDot -.-.-.-.
5 DashDotDot -..-..-..-..
6 LongDash DashDot보다 긴 선분의 반복
7 CircleDot Dot보다 큰 동그라미의 반복
8 Double 2중선
9 ThinBold 가는선 + 굵은선 2중선
10 BoldThin 굵은선 + 가는선 2중선
11 ThinBoldThin 가는선 + 굵은선 + 가는선 3중선

# 선 끝 모양

선 끝 모양을 나타냅니다.

libhwp에서 enum class인 HWPLineEndShape객체로 표현됩니다.

값 상수 설명
0 Round round
1 Flat flat

TIP

그림일 때는 "Round", 그리기 개체일 때는 "Flat"이 기본 값 입니다.

# 화살표 모양

화살표 모양을 나타냅니다.

libhwp에서 enum class인 HWPLineArrowShape객체로 표현됩니다.

값 상수 설명
0 None 모양 없음
1 Arrow 화살 모양
2 CancaveArrow 오목한 화살 모양
3 EmptyDiamond 속이 빈 다이아몬드 모양
4 EmptyCirlce 속이 빈 원 모양
5 EmptyRectangle 속이 빈 사가 모양
6 Diamond 속이 채워진 다이아몬드 모양
7 Circle 속이 채워진 원 모양
8 Rectangle 속이 채워진 사각 모양

# 화살표 크기

화살표 크기을 나타냅니다.

libhwp에서 enum class인 HWPLineArrowSize객체로 표현됩니다.

값 상수 설명
0 SmallSmall 작은-작은
1 SmallMiddle 작은-중간
2 SmallBig 작은-큰
3 MiddleSmall 중간-작은
4 MiddleMiddle 중간-중간
5 MiddleBig 중간-큰
6 BigSmall 큰-작은
7 BigMiddle 큰-중간
8 BigBig 큰-큰

# 그림자 정보

그림자 정보을 나타냅니다.

libhwp에서 HWPShadowInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 HWPShadowType type 그림자 종류
COLORREF 4 Color4Byte color 그림자 색깔
INT32 4 Int offsetX 그림자의 X 위치
INT32 4 Int offsetY 그림자의 Y 위치
UINT8 1 Short transparent 그림자의 투명도
- 4 - - 크기를 채우기 위한 byte
전체 길이 21

8 RightFront 오른쪽 앞

# 그리기 개체 글상자용 텍스트 리스트 헤더

그리기 개체 글상자용 텍스트 리스트 헤더을 나타냅니다.

libhwp에서 ListHeaderForHWPTextBox객체로 표현됩니다.

자료형 길이 타입 변수 설명
INT16 2 Int paraCount 문단 수
UINT32 4 HWPListHeaderProperty property 속성
HWPUNIT16 2 Int leftMargin 글상자 텍스트 왼쪽 여백
HWPUNIT16 2 Int rightMargin 글상자 텍스트 오른쪽 여백
HWPUNIT16 2 Int topMargin 글상자 텍스트 위쪽 여백
HWPUNIT16 2 Int bottomMargin 글상자 텍스트 아래쪽 여백
HWPUNIT 4 Long textWidth 텍스트 문자열의 최대 폭 [2]
INT32 4 Int textWidth 폼 모드에서 수정 가능 여부
WCHAR array n String? fieldName 필드 이름
전체 길이 가변 24 + n

  1. 초기엔 1, group할 때마다 하나씩 증가하고, ungroup할 때 마다 하나씩 감소합니다. ↩︎

  2. 보통 그리기 개체의 가로 크기와 동일 ↩︎
