# 타원 개체

Tag ID

HWPTAG_SHAPE_COMPONENT_ELLIPSE

Control ID

makeCtrlID('$', 'e', 'l', 'l')

# 데이터 표

타원 개체를 나타냅니다.

libhwp에서 HWPControlEllipse객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream n HWPCtrlHeaderGso header 개체 공통 속성
BYTE stream n2 HWPShapeComponentNormal shapeComopnent 그리기 개체 공통 속성
BYTE stream 60 HWPShapeComponentEllipse shapeComopnentEllipse 타원 개체 속성
BYTE stream n3 HWPTextBox? textBox 그리기 개체 글상자용 텍스트 속성
BYTE stream n4 HWPCaption? caption 캡션
BYTE stream n5 HWPCtrlData? ctrlData 컨트롤 데이터
전체 길이 가변 n + n2 + 60 + n3 + n4 + n5

# 타원 개체 속성

사각형 개체 속성을 나타냅니다.

libhwp에서 HWPShapeComponentRectangle객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 ...[1] property 속성
INT32 4 Int centerX 중심 좌표의 X 값
INT32 4 Int centerY 중심 좌표의 Y 값
INT32 4 Int axis1X 제1축 X 좌표 값
INT32 4 Int axis1Y 제1축 Y 좌표 값
INT32 4 Int axis2X 제2축 X 좌표 값
INT32 4 Int axis2Y 제2축 Y 좌표 값
INT32 4 Int startX start pos X
INT32 4 Int startY start pos Y
INT32 4 Int endX end pos X
INT32 4 Int endY end pos Y
INT32 4 Int startX2 start pos X2
INT32 4 Int startY2 start pos Y2
INT32 4 Int endX2 end pos X2
INT32 4 Int endY2 end pos Y2
전체 길이 60

# 타원/호 개체 속성의 속성

타원/호 개체 속성의 속성을 나타냅니다.

libhwp에서 HWPShapeComponentEllipseProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 Boolean _isRecalculateIntervalWhenChangingArc 호(ARC)로 바뀌었을 때, interval을 다시 계산해야 할 필요가 있는지 여부 [2]
bit 1 Boolean _isChangeArc 호(ARC)로 바뀌엇는지 여부
bit 2-9 Short _arcSort 호(ARC)의 종류

  1. HWPShapeComponentEllipseProperty ↩︎

  2. interval - 원 위에 존재하는 두 점 사이의 거리를 의미합니다. ↩︎
