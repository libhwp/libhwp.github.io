# 개체 공통 속성을 포함하는 컨트롤

extend type의 컨트롤은 종류를 나타내는 식별 기호로 32비트 ID가 사용된다. 컨트롤 코드가 큰 범주를 나타내는 식별 기호라고 한다면, 컨트롤 ID는 세부 분류를 나타내는 식별 기호인 셈입니다.

fun makeCtrlID(a: Char, b: Char, c: Char, d:Char) : Long = 
        (a.toInt().shl(24))
            .or(b.toInt().shl(16))
            .or(c.toInt().shl(8))
            .or(d.toInt())
            .toLong()

val coldef: Long = makeCtrlID('c', 'o', 'l', 'd') // 단 정의 컨트롤 ID

# 개체 공통 속성

개체 공통 속성을 나타냅니다.

libhwp에서 HWPCtrlHeaderGso객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 Long ctrlId ctrl ID
UINT32 4 ...[1] property 속성
HWPUNIT 4 Long yOffset 세로 오프셋
HWPUNIT 4 Long xOffset 가로 오프셋
HWPUNIT 4 Long width 오브젝트 폭
HWPUNIT 4 Long height 오브젝트 높이
INT32 4 Int zOrder z-order
HWPUNIT16 2 Int outerMarginLeft 오브젝트 왼쪽 여백
HWPUNIT16 2 Int outerMarginRight 오브젝트 오른쪽 여백
HWPUNIT16 2 Int outerMarginTop 오브젝트 위쪽 여백
HWPUNIT16 2 Int outerMarginBottom 오브젝트 아래쪽 여백
UINT32 4 Long instanceId 고유 아이디
INT32 4 Boolean preventPageDivide 쪽나눔 방지
on(1) / off(0)
WORD 2 - - 개체 설명문 글자 길이(len)
WCHAR array[len] 2xlen String explanation 개체 설명문 글자
전체 길이 가변 46 + (2 x len)

# 개체 공통 속성의 속성

개체 공통 속성의 속성을 나타냅니다.

libhwp에서 HWPGsoHeaderProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 Boolean _isLikeWord 글자 처럼 취급 여부
bit 2 Boolean _isApplyLineSpace 줄 간격 영향 여부
bit 3-4 HWPVertRelTo _vertRelTo 세로 위치 기준
bit 5-7 HWPRelativeArrange _vertRelativeArrange 세로 위치의 기준에 대한 상대적 배열 방식
bit 8-9 HWPHorzRelTo _horzRelTo 가로 위치 기준
bit 10-12 HWPRelativeArrange _horzRelativeArrange 가로 위치의 기준에 대한 상대적 배열 방식
bit 13 Boolean _isVertRelToParaLimit 세로 위치가 para일 때 세로 위치를 본문 영역으로 제한 여부
bit 14 Boolean _isAllowOverlap 다른 오브젝트와 겹침 허용 [2]
bit 15-17 HWPWidthCriterion _widthCriterion 오브젝트 폭 기준
bit 18-19 HWPHeightCriterion _heightCriterion 오브젝트 높이 기준
bit 20 Boolean _isProtectSize 세로 위치가 para일 때 크기 보호 여부
bit 21-23 HWPTextFlowMethod _textFlowMethod 텍스트가 오브젝트 주위를 어떻게 흘러갈지 지정
bit 24-25 HWPTextHorzArrange _textHorzArrange 오브젝트 좌/우 어느 쪽에 글 배치 옵션
bit 26-28 HWPObjectNumberSort _objectNumberSort 이 개체가 속하는 번호 범주
전체 길이 4

# 상대적 배열 방식

상대적 배열 방식을 나타냅니다.

libhwp에서 enum class인 HWPRelativeArrange객체로 표현됩니다.

값 상수 설명
0 TopOfLeft VertRelTo/HorzRelTo가 para 이면 top 그렇지 않으면 left
1 Center VertRelTo/HorzRelTo가 para 이면 center
2 BottomOrRight VertRelTo/HorzRelTo가 para 이면 bottom 그렇지 않으면 right
3 Inside VertRelTo/HorzRelTo가 para 이면 inside
4 OutSide VertRelTo/HorzRelTo가 para 이면 outside

# 텍스트가 오브젝트 주위를 어떻게 흘러갈지 옵션

텍스트가 오브젝트 주위를 어떻게 흘러갈지 옵션을 나타냅니다.

libhwp에서 enum class인 HWPTextFlowMethod객체로 표현됩니다.

값 상수 설명
0 Square bound rect를 따라
1 Tight 오브젝트의 outline을 따라
2 Through 오브젝트 내부의 빈 공간까지
3 TopAndBottom 좌, 우에는 텍스트를 배치하지 않음
4 BehindText 글과 겹치게 하여 글 뒤로
5 InFrontOfText 글과 겹치게 하여 글 앞으로

자료형 길이 타입 변수 설명
INT16 2 Int paraCount 문단 수
