# 구역 정의

Control ID

makeCtrlID('s', 'e', 'c', 'd')

# 데이터 표

구역 정의를 나타냅니다.

libhwp에서 HWPControlSectionDefine객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream 30 ...[1] header 구역 정의 헤더
BYTE stream 40 HWPPageDef pageDef 용지설정 정보
BYTE stream 26 HWPFootEndNoteShape footnoteShape 각주 모양 정보
BYTE stream 26 HWPFootEndNoteShape endnoteShape 미주 모양 정보
BYTE stream 12 HWPPageBorderFill bothPageBorderFill 양 쪽 정보
BYTE stream 10 HWPPageBorderFill oddPageBorderFill 홀수 쪽 정보
BYTE stream 10 HWPPageBorderFill oddPageBorderFill 짝수 쪽 정보
BYTE stream 10xn HWPBatangPageInfo batangPageInfo 바탕쪽 정보 [2]
BYTE stream n HWPCtrlData? ctrlData 컨트롤 데이터
전체 길이 140 + n

# 구역 정의 헤더

구역 정의 헤더를 나타냅니다.

libhwp에서 HWPCtrlHeaderSectionDefine객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 Long ctrlId 컨트롤 ID
UINT32 4 ... [3] property 속성
HWPUNIT16 2 Int columnGap 동일한 페이지에서 서로 다른 단 사이의 간격
HWPUNIT16 2 Int verticalLineAlign 세로로 줄맞춤 여부 [4]
HWPUINT16 21 Int horizontalLineAlign 가로로 줄맞춤 여부 [4:1]
HWPUNIT 4 Long defaultTabGap 기본 탭 간격 [5]
UINT16 2 Int numberParaShapeId 번호 문단 모양 ID
UINT16 2 Int pageStartNumber 쪽 시작 번호 [6]
UINT16 2 Int imageStartNumber 이미지 시작 번호 [6:1]
UINT16 2 Int tableStartNumber 표 시작 번호 [6:2]
UINT16 2 Int equationStartNumber 수식 시작 번호 [6:3]
UINT16 2 Int defaultLanguage 대표 Language [7] [8]
전체 길이 30

# 구역 정의 헤더 속성

구역 정의 헤더 속성을 나타냅니다.

libhwp에서 HWPSectionDefineHeaderProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 Boolean _isHideHeader 머리말을 감출지 여부
bit 1 Boolean _isHideFooter 꼬리말을 감출지 여부
bit 2 Boolean _isHideBatangPage 바탕쪽을 감출지 여부
bit 3 Boolean _isHideBorder 테두리를 감출지 여부
bit 4 Boolean _isHideBackground 배경을 감출지 여부
bit 5 Boolean _isHidePageNumberPosition 쪽 번호 위치를 감출지 여부
bit 8 Boolean ... [9] 구역의 첫 쪽에만 테두리 표시 여부
bit 9 Boolean ... [10] 구역의 첫 쪽에만 배경 표시 여부
bit 16-18 HWPTextDirection _textDirection 텍스트 방향
bit 19 Boolean _isHideEmptyLine 빈 줄 감춤 여부
bit 20-21 Boolean ... [11] 구역 나눔으로 새 페이지가 생길 때의 페이지 번호 적용 여부
bit 22 Boolean _isApplyWongoji 원고지 정서법 적용 여부

# 용지 설정

Tag ID

HWPTAG_PAGE_DEF

용지 설정을 나타냅니다.

libhwp에서 HWPPageDef객체로 표현됩니다.

자료형 길이 타입 변수 설명
HWPUNIT 4 Long paperWidth 용지 가로 크기
HWPUNIT 4 Long paperHeight 용지 세로 크기
HWPUNIT 4 Long leftMargin 용지 왼쪽 여백
HWPUNIT 4 Long rightMargin 용지 오른쪽 여백
HWPUNIT 4 Long topMargin 용지 위쪽 여백
HWPUNIT 4 Long bottomMargin 용지 아래쪽 여백
HWPUNIT 4 Long headerMargin 용지 머리말 여백
HWPUNIT 4 Long footerMargin 용지 꼬리말 여백
HWPUNIT 4 Long gutterMargin 용지 제본 여백
UINT32 4 HWPPageDefProperty property 속성
전체 길이 40

# 용지 설정 속성

용지 설정 속성을 나타냅니다.

libhwp에서 HWPPageDefProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 HWPPaperDirection _paperDirection 용지 방향
bit 1-2 HWPMakingBookMethod _makingBookMethod 제책 방법

# 용지 방향

용지 방향을 나타냅니다.

libhwp에서 enum class인 HWPPaperDirection객체로 표현됩니다.

값 상수 설명
0 Portrait 좁게
1 Landscape 넓게

# 제책 방법

제책 방법을 나타냅니다.

libhwp에서 enum class인 HWPMakingBookMethod객체로 표현됩니다.

값 상수 설명
0 OneSideEditing 한쪽 편집
1 BothSideEditing 맞쪽 편집
2 BackFlip 위로 넘기기

# 각주/미주 모양

Tag ID

HWPTAG_FOOTNOTE_SHAPE

용지 설정을 나타냅니다.

libhwp에서 HWPPageDef객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT32 4 ... [12] property 속성
WCHAR 2 String? userSymbol 사용자 기호
WCHAR 2 String? beforeDecorativeLetter 앞 장식 문자
WCHAR 2 String? afterDecorativeLetter 뒤 장식 문자
UINT16 2 Int startNumber 시작 번호
HWPUNIT16 2 Long divideLineLength 구분선 길이
HWPUNIT16 2 Int divideLineTopMargin 구분선 위 여백
HWPUNIT16 2 Int divideLineBottomMargin 구분선 아래 여백
HWPUNIT16 2 Int betweenNotesMargin 주석 사이 여백
UINT8 1 HWPBorderType divideLineSort 구분선 종류
UINT8 1 HWPBorderThickness divideLineThickness 구분선 굵기
COLORREF 4 Color4Byte divideLineColor 구분선 색상
전체 길이 26

# 각주/미주 모양 속성

각주/미주 모양 속성을 나타냅니다.

libhwp에서 HWPFootnoteShapeProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0-7 HWPNumberShape _numberShape 번호 모양
bit 8-9 HWPFootnoteDisplayMethod _footnoteDisplayMethod 한 페이지 내에서 각주를 다단에 위치시킬 방법
bit 8-9 HWPEndnoteDisplayMethod _endnoteDisplayMethod 한 페이지 내에서 미주를 다단에 위치시킬 방법
bit 10-11 HWPNumberingMethod _numberingMethod numbering
bit 12 Boolean _isDisplayWithSupscript 각주 내용 중 번호 코드의 모양을 위 첨자 형식 여부
bit 13 Boolean _isContinueFromText 텍스트에 이어 바로 출력 여부

# 각주 다단 위치 방법

각주 다단 위치 방법을 나타냅니다.

libhwp에서 enum class인 HWPFootnoteDisplayMethod객체로 표현됩니다.

값 상수 설명
0 EachColumn 각 단마다 따로 배열
1 AllColumn 통단으로 배열
2 RightColumn 가장 오른쪽 단에 배열

# 미주 다단 위치 방법

미주 다단 위치 방법을 나타냅니다.

libhwp에서 enum class인 HWPEndnoteDisplayMethod객체로 표현됩니다.

값 상수 설명
0 DocumentEnd 문서의 마지막
1 SectionEnd 구역의 마지막

# Numbering

Numbering을 나타냅니다.

libhwp에서 enum class인 HWPNumberingMethod객체로 표현됩니다.

값 상수 설명
0 Continue 앞 구역에 이어서
1 Restart 현재 구역부터 새로 시작
2 RestartAtEachPage 쪽마다 새로 시작(각주 전용)

# 쪽 테두리/배경

Tag ID

HWPTAG_PAGE_BORDER_FILL

쪽 테두리/배경을 나타냅니다.

libhwp에서 HWPPageBorderFill객체로 표현됩니다.

자료형 길이 타입 변수 설명
UINT 4 HWPPageBorderFillProperty property 속성
HWPUNIT16 2 Int leftGap 왼쪽 간격
HWPUNIT16 2 Int rightGap 오른쪽 간격
HWPUNIT16 2 Int topGap 위쪽 간격
HWPUNIT16 2 Int bottomGap 아래쪽 간격
UINT16 2 Int borderFillId 테두리/배경 아이디
전체 길이 12

# 쪽 테두리/배경 속성

쪽 테두리/배경 속성을 나타냅니다.

libhwp에서 HWPPageBorderFillProperty객체로 표현됩니다.

범위 타입 변수 설명
bit 0 HWPPositionCriterion _positionCriterion 위치 기준
bit 1 Boolean _isIncludeHeader 머리말 포함
bit 2 Boolean _isIncludeFooter 꼬리말 포함
bit 3-4 HWPFillArea _fillArea 채울 영역

# 위치 기준

위치 기준을 나타냅니다.

libhwp에서 enum class인 HWPPositionCriterion객체로 표현됩니다.

값 상수 설명
0 MainText 본문 기준
1 Paper 종이 기준

# 채울 영역

채울 영역을 나타냅니다.

libhwp에서 enum class인 HWPFillArea객체로 표현됩니다.

값 상수 설명
0 Paper 종이
1 Page 쪽
2 Border 테두리

# 바탕쪽 정보

바탕쪽 정보를 나타냅니다.

libhwp에서 HWPBatangPageInfo객체로 표현됩니다.

자료형 길이 타입 변수 설명
BYTE stream 16 ListHeaderForBatangPage listHeader 바탕쪽 정보 리스트 헤더
BYTE stream n HWPParagraphList paragraphList 문단 리스트
전체 길이 16 + n
